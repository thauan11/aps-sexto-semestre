Sobre

Algumas informações sobre o desenvolvimento do projeto

Esse site é um projeto cujo objetivo é possibilitar a alteração de imagens utilizando Python. Foi criado utilizando essencialmente as bibliotecas Flask e OpenCV. O projeto visa possibilitar para o usuário final a manipulação de imagens sem a necessidade de instalar nenhum programa.

O código do site é encontrado no GitHub, para fins educacionais, o site pode ser acessado e manipulado livremente. Para obter mais informações sobre nossa abordagem, consulte nosso GitHub e experimente você mesmo.

Ideia

O projeto nasceu da necessidade de se criar um programa que fizesse a manipulação de imagens, como aplicação de filtro, reconhecimento facial ou conversão de imagem em texto. Foi projetado a criação de um site, pois dessa forma não haveria necessidade do usuário final realizar instalação de aplicativo, sendo mais seguro e prático.

Muitos programas atualmente estão migrando para nuvem, e como principal fonte de estudo, foi realizado uma pesquisa sobre o funcionamento do Azure (Microsoft), que possui o Cloud Azure, onde a Microsoft visa atender a necessidade de seu cliente e prove uma forma prática dos usuários conseguirem trabalhar em nuvem, sendo uma das suas principais aplicações o Windows Virtual Desktop, onde é executado um computador totalmente funcional em um navegador.

Sendo essa a maior inspiração, provando ser possível desenvolver uma aplicação capaz que conseguiria executar no back-end de um site.

Projeto

Após definir o tipo de serviço, iniciou-se a busca e desenvolvimento de um layout que combinasse com o projeto. Utilizado como referência, o NumPy foi definido como layout modelo, por possuir cores agradáveis e um design mais minimalista.

De início o back-end começou a ser estruturado através da biblioteca Django, o problema encontrado mais a frente no projeto, para as ideias que foram decididas, o Django tornou-se complicado de manipular, mais do que o processamento de imagem, devido a isso, uma nova estrutura foi constituída, utilizando a biblioteca Flask. A decisão de utilizar o Flask foi devido a sua facilidade de manipulação e aplicação.

Para manipulação de imagem, a biblioteca escolhida havia sido Pillow, pois era de simples implementação. Ao longo do projeto, foi instaurado um novo desafio, tornou-se a ser obrigatório o uso do OpenCV como biblioteca principal para manipulação e processamento da imagem.

Com desing pronto e linguagem escolhida, seguiu-se para ultima etapa, pegar todos os elementos, unifica-los e fazer funcionar. Com Flask, montamos as páginas e suas funções, utilizando Javascript, foi possivel exibir a imagem, para que o usuario saiba que imagem será carregada e manipulada, HTML e CSS foi construido todo o layout.
